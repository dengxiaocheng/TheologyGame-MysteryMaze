# 通向奥秘 / Path to Mystery

基于拉纳神学的迷宫探索游戏。

**在线试玩**: [https://dengxiaocheng.github.io/TheologyGame-MysteryMaze/](https://dengxiaocheng.github.io/TheologyGame-MysteryMaze/)

## 技术栈
- 纯前端 HTML/CSS/JavaScript
- Canvas 渲染
- 无需构建工具，直接浏览器打开即可

## 开发
```bash
# 本地运行
python3 -m http.server 8000
# 访问 http://localhost:8000
```
